Over all

  • in a $n*m$ matrix
  • creat a two-part graph with n+m nodes (sometimes an extra S and an extras T)
  • if $matrix[i][j] == 1$(or similar), connect verdex $i$ & $n + j$

Mainly used in

  • of course a matrix
  • each unit has two states (at least O(1) states)
  • (maybe) need to do something with those special unit
